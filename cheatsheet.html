TIPS & TRICKS

  • Show LEDs-blok: Laat patronen of tekens zien op de micro:bit om visuele feedback te geven. Probeer verschillende patronen uit.
  • Loops: Gebruik de “forever” loop voor herhalende taken. Dit bespaart tijd en maakt je code korter.
  • Variabelen: Bewaar gegevens in variabelen om ze later weer te gebruiken. Dit maakt je code flexibeler en makkelijker te beheren.
  • Inputs en Outputs: Leer wat de micro:bit kan ontvangen en uitsturen om meer interactie en functies toe te voegen aan je projecten.
  • For-loops: Gebruik “for” loops om automatisch een reeks acties uit te voeren, zoals het tonen van nummers. Dit bespaart je veel handmatig werk.
  • Vergelijkingen: Gebruik symbolen zoals `>`, `<`, en `=` om voorwaarden te testen in je programma. Dit helpt bij het maken van beslissingen in je code.
  • Boolean logica: Combineer voorwaarden met “and” en “or” om complexere beslissingen te maken in je programma.
  • Test je code vaak: Controleer je programma regelmatig tijdens het schrijven om problemen snel te vinden en op te lossen.
  • Documenteer je code: Voeg opmerkingen toe aan je code om uit te leggen wat elk deel doet. Dit helpt je later om je code beter te begrijpen.
